HTML and CSS skills may not be my most technologically advanced skills to showcase, but I think that it utilizes a more creative format to showcase what I can do as a programmer. I started this project by laying out my designs and my inspirations through website prototyping tools to design exactly what I wanted: a showcase of myself. - Utilizing HTML, CSS and Bootstrap to create a custom website

- Designed a creative and colorful alternative to a physical portfolio

HTML, CSS, Bootstrap


My focus in Information Technology was never really on web design, so when I decided to create a portfolio website from scratch using HTML, CSS and bootstrap, it was a little bit out of my area of expertise. However, it was a challenge I wanted to accept and the less. This was my first attempt at using bootstrap and I found that it really did speed along a lot of the development processes while still creating the unique style that I was going for. I found it very freeing to work with HTML to create my own website instead of using a website builder like WIX. I also found that a HTML and CSS webpage shows a lot more character and technical skill than the premade templates that can be found online. The undertaking of this project was no small task and did require a lot of time to complete, but I believe that in game development you need to put in the work to succeed. This website is a prime example of how dedicated I am to my work.
