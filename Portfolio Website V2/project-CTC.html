Crossing The Cosmos

Crossing The Cosmos is a high stakes hardcore resource management game created for the "Finish Your Game" Game Jam.

Fly across space while managing the SS PROMETHEUS to keep your resources high and your crew healthy. It will be a long and deadly voyage to a new home in New Terra so make sure you are ready captain!

Key Features

Crossing The Cosmos was original intended as a weeklong game jam but turned into a project that took us a couple months as we created more interesting mechanics and produced a full game experience with lots of polish until we were happy with the final project. We then joined the “Finish Your Game” game jam with our final game! The project was super interesting and helpful to me personally for learning game design aspects to design the game from scratch as well and working with more junior project members and learning how best to mentor them.

Major roles and tasks done on this project include,

- Game design
- System architecture and design
- UI/UX design
- Randomization systems
- Balancing and difficulty scaling
- Creative writing

2D Gameplay


Crossing The Cosmos uses many different 2D gameplay systems that I have since become very acquainted with such as AI pathing for the moving characters on screen, both frontend and backend systems for the UI interactions, polishing animations for UI and the main ship to breath life into the game.
