Raspberry Pi Web Server

I have always wanted to be more involved and in control of my web hosting. In 2019, I bought my first Raspberry Pi to learn more about the hardware and automation applications. One of the first things I ever wanted to do with a Raspberry Pi was figure out how to host my own web server. I wanted to be to able display my projects onto the web such as my portfolio, small websites, and future web development projects so that others may access them as well.

** As of late 2021 this website is now hosted by HostGator at "http://teenytinygames.net" instead of "http://teenytinygames.webhop.me" to boost loading speeds that were not possible on a home network!**

Key Features

Creating my web server on the Raspberry Pi covered a lot of major skills from networking solutions to Linux commands. I did a fair amount of research to understand the ins and outs of hosting my own web servers and while this server is not very scalable, I think it meets my needs perfectly.


My major roles and tasks done on this project include:

- Utilized Linux commands to prepare web servers
- Researched and implemented networking solutions for web hosting
- Accessed routers and services to re-route port access to my website

Software Hard Skills


Working with a Raspberry Pi meant being able to have creativity with touch of technicality which was incredibly exciting for me. Setting up a web server on the Raspberry Pi or any sort of device can be a little bit challenging to those breaking into the necessary technologies. While setting up the web server, I needed to do a lot of research on how my idea was going to play out. I used a website called No-IP to redirect the website URL on to my IP address. From there, I had my router redirect port 80, the most common port for websites over to my Raspberry Pi’s IP. On the server side of the Raspberry Pi, I need to set up software for No-IP so that it could access the website. Afterwards, I installed Apache to host the website within the Raspberry Pi. However, both softwares did require the use of UNIX commands to install and set up. Luckily, I had my technical schooling handy to be able to understand and complete the commands for the software I needed.
