Explore procedurally generated lands and dungeons in this styled survival game where you must feed not only yourself, but your adorable dodo followers!

Key Features

Dodo Town is one of my currently active projects which happens to be a spin-off of one of my other works, The Mountain. I wanted to explore some of the gameplay mechanics from the previous game to build the foundation of a more improved first-person survival game with my newfound experience. This project is heavily focused on survival elements of the player, NPCs and the land with procedural generation being a core element. Much of what has been put into the project has been developed from building upon what I learned from other projects.


My major roles and tasks done on this project include:

- Developed advanced programming solutions in the Unity engine
- Reduced and filtered game design concepts from previous projects
- Utilized advanced programming techniques in manipulating 3D objects
- Developed procedural designs in programming experiments
- Created visual dioramas to build a clear and concise art and style direction

Reducing Game Design & Mechanics


Dodo Town was developed as a condensation of earlier projects. I spend a lot of work reducing the core concept by reducing it into what worked and what didn’t work from these older projects. I keep a lot of the mechanics that were more generalized and that were tried-and-true. Using the rest of my time on the project to build uniqueness with the established basic features. The Mountain inspired me to continue working on procedural generation designs. I also found that implementing a base camp mechanic in these genres gives a player a place to build and call “home”. I did remove older mechanics and horror elements of the previous game to make a more light-hearted environment.



Procedural Generated Terrain & Hex Designs


For this project a procedurally generated hexagonal terrain was one of my main focusses. I have always found the idea simplistic objects like the Minecraft cube to be very interesting. I wanted to try a new take on the style. Hexes are commonly associated with strategy games, so there is a bit of a hurdle to overcome that mental association of that idea when developing for a non-strategic game. There’s a lot of technical work going on behind the scenes to get the hexes to work as spawning tens of thousands of hexagonal game objects is not really an option due to performance requirements. The core idea focuses on the spawning of models then merging them together so that they can be used on an individual basis as well as a single rendering mesh. The combined mesh does require extra work to be able to merge textures and colors after it is developed but does the job efficiently saving framerate.



Creating Concept Art, Style & Dioramas


When starting out with Dodo Town, I wanted to have a solid foundation of the visual style that I wanted to pursue. I created dioramas of interesting concept-art pieces to base the game visuals around. This gave me an idea of what I wanted to focus towards instead of an ever-changing concept in my mind. I found this really helped me decide how I wanted the terrain, the lighting and the color scheme to look.
