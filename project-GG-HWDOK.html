Homeworld: Deserts of Kharak - EOS Patch

Homeworld: Deserts of Kharak is a real-time strategy video game developed by Blackbird Interactive.

Deserts of Kharak was originally released as a steam only game, our team was in charge of porting the game to the Epic Games Store with mirrored crossplay functionality as well as additional bugs and other requested features.

Key Details

Working alongside Gearbox staff, I had the privilege of contributing to the re-release of Homeworld: Deserts of Kharak onto the Epic Games Store with EOS integration. Using the steam integrations as a guide we navigated through the C# code base to deliver our update and support a promotional campaign for an upcoming installment in the franchise. My portion of work heavliy focused on designing and enginnering a UI system capable of parsing company web JSON data and presenting it in an appealing and user-friendly format.

My major roles and tasks done on this project include:

Worked to integrate EOS achievements, game presence, cross-platform multiplayer, and leaderboards.
Fabricated a UI system that parsed web JSON data and displayed the information in an appealing form.
Coordinated with other team members to resolve C# performance issues and fix legacy bugs.

Epic Games Online Integrations


During my role on the Homeworld: Deserts of Kharak, I aided in integrating the game menus and gameplay aspects with the new EOS service. This encompassed setting up preprocessor directives for Steam and EOS and connecting menus for achievements, game presence, cross-platform multiplayer, and leaderboards. The process involved setting up connections and data calls between the menus and backend services.



UI Features


One of the key features I worked on was the creation of a UI system that could parse JSON data from the company website and display it in an appealing and user-friendly format. This involved designing and engineering the system to handle the data and present it in a way that was easy for players to understand as well as being easy to navigate out of to ensure an enjoyable user experience. I also worked to ensure that the system was scalable and could be easily updated as new data became available through the web portal.



Multiplayer Debugging


In addition to adding the new EOS features I dedicated significant effort to testing and helping refine the game's multiplayer experience with the crossplay between the two systems. This involved extensive testing using multiple clients to compare and fix gameplay issues between Steam and EOS servers as we migrated services. identify and resolve cross-platform bugs between Steam and EOS systems. Additionally, I collaborated with my team members as we diagnosed and fixed large scale multiplayer bugs, enhancing overall performance and reliability for players.
